Version: 1.0

This directory contains the only official brand assets for JivniCare.

All designers, developers, AI agents, automation tools, and implementation systems must use only the assets and rules defined in this directory.

# Image Rules

Never stretch.

Never squash.

Never rotate.

Never skew.

Never recolor.

Never crop.

Never add shadows.

Never add outlines.

Never add glow effects.

Never change opacity.

Never modify proportions.

Never separate icon and wordmark.

Never recreate the logo.

---

# Background Rules

Transparent PNG must remain transparent.

SVG is the preferred format.

Never use preview images in production.

Never use screenshots from documentation.

---

# AI Implementation Rules

If an implementation agent cannot determine which asset should be used,

it must stop,

read this Brand System,

and use the asset mapping defined in this directory.

Guessing is prohibited.
